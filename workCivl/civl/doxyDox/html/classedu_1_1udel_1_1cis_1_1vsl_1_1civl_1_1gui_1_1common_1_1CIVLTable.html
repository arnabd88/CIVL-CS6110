Detailed Description

This class is used to ensure that the first column of the CIVLTable(JTable) is not editable, as we do not want users to be able to change the option names in the table where selected options are displayed.

In addition to this, it contains a custom cell renderer and a custom cell editor to override undesirable JTable behavior.
