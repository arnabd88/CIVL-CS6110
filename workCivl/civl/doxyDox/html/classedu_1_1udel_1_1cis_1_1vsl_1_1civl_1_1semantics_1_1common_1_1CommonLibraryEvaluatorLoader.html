Obtains the library evaluator of the given name.

Given the same name, it will always return the same instance of the library evaluator of that name.

Parameters
nameThe name of the library whose evaluator is to be obtained.
primaryEvaluatorThe CIVL evaluator of the system.
modelFacotryThe model factory of the system.
symbolicUtilThe symbolic utility for manipulations of symbolic expressions.
symbolicAnalyzerThe symbolic analyzer used in the system.
Returns
The library evaluator of the given name.
