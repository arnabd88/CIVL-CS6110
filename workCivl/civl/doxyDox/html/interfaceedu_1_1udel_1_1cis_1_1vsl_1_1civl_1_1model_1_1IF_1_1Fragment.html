Detailed Description

A fragment is a sequence of statements.

It has a "pointer" to the start location and a set of statements as its final statements which should all enamate from the same location.

Author
Manchun Zheng

void edu.udel.cis.vsl.civl.model.IF.Fragment.addGuardToStartLocation ( Expression  guard,
ModelFactory  factory 
)

Add a specified guard to the all statements of the start location.

If a statement has an existing guard, then it will have a new guard which is a conjunction of the both. This method is used for translating a when statement, where it adds the guard of the when statement to the start location of its body fragment.

Parameters
guardThe guard that is to be combined with
factoryThe model factory that provides some helper methods that are useful in checking if an expression is True.
