Detailed Description

This is the guard expression of CIVL for loops ($for).

It has the form of (i, j, k, ... : dom), where i, j, k ... are iteration variables and dom is the domain expression. The number of iteration variables should be equal to the dimension of the domain. A domain guard expression is evaluated to be true if and only if there exists a subsequent element of (i, j, k, ...) in the domain.
