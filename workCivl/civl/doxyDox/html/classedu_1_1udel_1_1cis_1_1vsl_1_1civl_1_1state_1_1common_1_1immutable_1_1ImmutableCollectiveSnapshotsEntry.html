ImmutableMonoState  monoState,
Expression  assertion 
)
inline

Insert a new ImmutableMonoState belonging to a DIFFERENT (if the process is existed, an error will be reported) process.

Returns the number of the stored ImmutableMonoState after the insertion. If the insertion completes the entry, it changes the result which will be returned by isComplete().

The PID of the MonoState must be in involvedProcesses() and never be recorded i.e. isRecorded(PID) returns false and isComplete() returns false.
