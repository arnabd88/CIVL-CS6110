Obtains the library executor of the given name.

Given the same name, it will always return the same instance of the library executor of that name.

Parameters
nameThe name of the library whose executor is to be obtained.
primaryExecutorThe CIVL executor of the system.
modelFacotryThe model factory of the system.
symbolicUtilThe symbolic utility for manipulations of symbolic expressions.
symbolicAnalyzerThe symbolic analyzer used in the system.
Returns
The library executor of the given name.
Exceptions
LibraryLoaderExceptionIf the library executor of the given name cannot be found.
