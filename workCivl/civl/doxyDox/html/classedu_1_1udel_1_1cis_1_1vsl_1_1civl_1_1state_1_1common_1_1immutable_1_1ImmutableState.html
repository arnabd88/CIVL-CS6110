Detailed Description

Implementation of State based on the Immutable Pattern.

This class is not entirely immutable; it has certain fields such as onStack and depth used by the depth first search algorithm which can be modified. But it has an "observationally immutable core" consisting of the path condition, dynamic scopes, and process states. While these can also change in certain restricted ways, these changes are not visible to a user going through the State interface.

The path condition and equals methods are based solely on the observationally immutable core.
