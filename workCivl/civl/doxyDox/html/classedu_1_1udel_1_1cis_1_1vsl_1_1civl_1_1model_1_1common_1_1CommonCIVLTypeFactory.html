int edu.udel.cis.vsl.civl.model.common.CommonCIVLTypeFactory.getHeapFieldId ( CIVLType  type)
inline

Returns the type of the heap field of the given index.

A CIVL model has its unique heap type, which has type of tuples of arrays of arrays of type, determined by malloc statements and handle types appear in the source program. For examples, given the following program,

int main(){
  int* p = malloc(sizeof(int)*4);
  $gcomm gcomm = $gcomm_create(...);
  ...
}

the heap type of the model of this program will be (int[][], gcomm[][]).

Parameters
typethe type which is a field of the heap type
Returns
the ID of the type in the heap
