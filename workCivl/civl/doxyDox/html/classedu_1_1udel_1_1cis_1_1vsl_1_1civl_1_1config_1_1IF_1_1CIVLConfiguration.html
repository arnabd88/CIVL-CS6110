Detailed Description

A CIVLConfiguration object encompasses all the parameters used to configure CIVL for the execution of one or more tasks.

It provides methods to get and set these parameters. The types of the parameters are all simple Java types, such as boolean or int, so this class does not use any other CIVL classes
